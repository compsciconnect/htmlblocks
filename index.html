What is a webpage?

Webpages are made out of code. The code that creates the google.com page looks something like this: http://www.zpravy.net/misc/google-redesign/homepage-source.htm.

What is code?

Code is a set of instructions written in a particular programming language. Just like the instructions for baking a cake might be written in English or Spanish, different programming languages can be used to write instructions that produce the same result.

What is HTML?

The universal programming language for the web is HTML (HyperText Markup Language). In order to format the results of the instructions we write using HTML (in order to make the instructions in our cookbook look prettier) we use CSS (cascading style sheets).
